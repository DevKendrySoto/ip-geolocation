# ip-geolocation

Este proyecto es una aplicación web de práctica diseñada para demostrar cómo consumir una API de geolocalización de IP y cómo implementar un botón de modo oscuro (Dark Mode) utilizando HTML, CSS y JavaScript. La aplicación permite a los usuarios ingresar una dirección IP y obtener información detallada sobre su geolocalización, como la ciudad, el país y las coordenadas. Además, cuenta con un botón que permite cambiar el tema de la interfaz entre un modo claro y un modo oscuro, mejorando así la experiencia del usuario en diferentes condiciones de iluminación.

## Características

- Consumo de API para obtener la geolocalización de una IP.
- Visualización de la información de geolocalización obtenida de la API.
- Botón para cambiar entre modo claro y modo oscuro.
- Diseño responsivo que se adapta a diferentes tamaños de pantalla.

## Tecnologías Utilizadas

- HTML: Estructura de la página web.
- CSS: Estilos para el diseño de la página, incluyendo los temas claro y oscuro.
- JavaScript: Lógica para consumir la API, manejar eventos del usuario y cambiar los temas.

### Instrucciones de Instalación

1. Clona el repositorio en tu máquina local utilizando Git:

```bash
git clone https://github.com/DevKendrySoto/ip-geolocation.git
```
2. Navega al directorio del proyecto:
```bash
cd tu-repositorio
```
3. No es necesario instalar dependencias adicionales para ejecutar este proyecto, ya que solo utiliza HTML, CSS y JavaScript puro.

Para ejecutar la aplicación, simplemente abre el archivo `index.html` en tu navegador web. Si deseas utilizar un servidor local para ejecutar el proyecto, puedes utilizar la extensión "Live Server" en Visual Studio Code o ejecutar un servidor simple con Node.js usando el paquete `http-server`:

Navega a `http://localhost:8080` en tu navegador para ver la aplicación en funcionamiento.

## Uso de la Aplicación

1. Ingresa una dirección IP en el campo de texto proporcionado en la página.
2. Haz clic en el botón "Obtener Geolocalización" para ver la información de geolocalización de la IP.
3. Utiliza el botón "Cambiar a Modo Oscuro" o "Cambiar a Modo Claro" para alternar entre los temas de la aplicación.
